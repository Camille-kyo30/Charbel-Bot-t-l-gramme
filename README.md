## 📌 À propos

**Master Charbel Bot V2** est une version améliorée du bot Telegram créé par **RONALD SORY ホ**, modernisée et enrichie par **Master Charbel**.

### ✨ Nouvelles fonctionnalités V2

| Fonctionnalité | Description |
|---|---|
| 🎨 **Interface améliorée** | Messages plus clairs avec emojis et Markdown |
| 📊 **Statistiques en temps réel** | Comptage des utilisateurs et messages |
| 🎯 **Menu interactif** | Boutons inline pour une navigation facile |
| 🤖 **IA intégrée** | Réponses intelligentes via GPT-4o |
| 👑 **Système d'administration** | Commandes réservées aux admins |
| 💾 **Base de données** | SQLite pour mémoriser les utilisateurs |
| ⏰ **Tâches programmées** | Cron jobs pour automatisations |
| 🎨 **Logs colorés** | Console plus lisible et professionnelle |
| 🌐 **API de statut** | Endpoint `/status` pour surveiller le bot |
| 🔒 **Sécurité renforcée** | Gestion sécurisée des tokens |
